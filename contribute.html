Contribute

Anyone can submit ideas, content, and revisions to this website. You do not need to know anything about website development, but you will need to learn a few concepts related to Github which provides the website hosting and issue tracking features.

  • Anyone contributing must follow the Code of Conduct
  • Anyone contributing agrees to the license

If you don’t know anything about building websites, that’s fine. Here are the basic steps to follow:

Step 1

Sign up / sign in to Github. Git is a version control system designed for computer programmers. Github is an online platform that uses Git, and can help with all kinds of collaborative projects.

Step 2

Create an Issue describing your idea for new content or content to change. The issue tracker provides a forum for the community to discuss.

Next Steps

You can stop after step 2 if you just want to participate and discuss ideas. The next steps are if you want to create or edit content on this website.

Step 3

Create a Fork of the website. This copies the website content to your Github account. You can then freely implement the changes discussed in the issue you opened in Step 2 without worry of messing up the original website.

  • All website content is written in Markdown, an easy-to-learn syntax that does not require expert programming skills!

Step 4

Submit a Pull Request. The project maintainers will review your changes. After any discussion, you may be asked to make revisions. After revisions, the final version of your changes will be pulled into the website as appropriate.
